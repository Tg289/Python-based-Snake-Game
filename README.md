# Snake Game

## Description
A classic Snake game built using Python and Pygame.  
Control the snake with arrow keys, eat food to grow, and avoid colliding with walls or yourself.

## Features
- Arrow key controls
- Score tracking
- Growable snake
- Restart after losing

## Tech Stack
- Python 3.x
- Pygame
